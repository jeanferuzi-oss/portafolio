# Portafolio
Este proyecto predice el nivel de PIB de países usando datos del Banco Mundial. Se divide en tres etapas: limpieza e imputación de datos, reducción de dimensiones con PCA y modelación con algoritmos de clasificación. El objetivo es transformar indicadores complejos en conocimiento estratégico sobre el desarrollo global.

El trabajo de Ciencia de datos para economía busca crear un sistema capaz de predecir el nivel de riqueza de los países usando cifras del Banco Mundial. La idea es convertir datos difíciles en información útil para entender cómo crecen las naciones. Todo el estudio sigue un plan técnico dividido en tres partes principales. Se usó la plataforma GitHub para organizar el código en diferentes carpetas llamadas ramas. En la primera parte, revisamos los datos con mucho cuidado para asegurar que fueran de buena calidad. Si a una variable le faltaba menos del quince por ciento de su información, completamos los huecos, pero si faltaba más, borramos esa variable para no dañar los resultados finales del modelo. También dividimos el nivel de riqueza en cinco grupos: Bajo, Medio Bajo, Medio, Medio Alto y Alto. Esto ayuda a que el sistema aprenda a clasificar mejor a cada país de manera sencilla. En la segunda parte, como había muchísima información de distintos tipos, usamos una técnica llamada Análisis de Componentes Principales. Antes de esto, igualamos todas las escalas de los números para que fueran comparables. Este paso sirvió para achicar el grupo de datos original y quedarse solo con lo más importante. Logramos guardar entre el setenta y el noventa por ciento de la información útil, quitando las cosas que no servían o que se repetían entre las que teníamos al principio. Por último, en la tercera fase, usamos estos datos limpios y resumidos para enseñar a programas de computadora a adivinar el nivel económico de cada lugar. Todo este camino está escrito de forma clara para que cualquiera pueda repetirlo. El resultado final se guardó en la parte principal del repositorio después de juntar con éxito todos los avances de cada etapa del proyecto de manera ordenada.
